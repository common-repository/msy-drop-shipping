== Installation ==

1. Go to WP Dashboard > Plugins > Add New > Upload - and click Activate
2. Once activated, navigate to WP Dashboard > MSY
3. Enter your info into the fields and complete your registration.
4. That's it! Now it's time to enjoy MSY <3
